Manifesto JSON Schema APIs

JSON Schema is a specification for a JSON-based format for defining the structure of JSON data. Like XML just for JSON. We are uncovering better ways of developing API by using the JSON Schema. Through this work we have come to value:

  • documentation
  • highly decoupled from the code resource representation
  • readable, easy accessible
  • machine readable FTW!
